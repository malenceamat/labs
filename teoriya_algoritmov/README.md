## Как скомпилировать?
1. открываем консоль в папке с проектом
2. в консоль пишем: g++ имя_файла -o test
3. запускаем ./test
4. Либо воспользуйтесь онлайн компилятором, к примеру -> https://www.onlinegdb.com/online_c++_compiler
